== Installation ==

This section describes how to install the plugin and get it working.


Follow below instruction to setup 


1. Unzip the `library-book-search.zip` and Upload  to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Place `<?php do_action('library-listing-searching'); ?>` in your templates

4. Add shortcode [library-listing-searching] into any page/post for access all Library Books Listing with Searching
